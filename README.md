# NoteHub

A simple notes application for viewing, searching, creating, and deleting notes.  
Built with React + TypeScript and powered by the NoteHub backend API.

## Features

- Fetch and display notes from the API
- Debounced search (to avoid requests on every keystroke)
- Pagination (server-side)
- Create a new note in a modal window
- Delete a note
- Form validation (Formik + Yup)
- Styling with CSS Modules

## Tech Stack

- React + TypeScript (Vite)
- Axios (HTTP requests)
- TanStack Query (server state)
- Formik + Yup (forms and validation)
- React Paginate (pagination UI)
- use-debounce (debounced search)
- modern-normalize (CSS normalization)
- Vercel (deployment)
